## Features
- Smart best-selector detection (ID ‚Üí name ‚Üí class ‚Üí CSS ‚Üí XPath)
- Multiple XPath variants (absolute, relative, text, contains)
- Framework-ready selectors (Playwright, Cypress, Selenium, Java, Python, C#)
- Drag-and-drop floating selector panel
- Mini selector popup for quick copy
- Clean UI inspired by Chrome DevTools

## Installation
Download the extension and load it in Chrome:
1. Go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load Unpacked**
4. Select the extension folder
