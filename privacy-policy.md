Universal Selector Tool is a Chrome extension designed to help users inspect webpage elements and generate CSS selectors, XPath, and automation-ready locators.

This extension values your privacy.
We do not collect, store, transmit, or share any personal information.

1. Information We Do Not Collect

The extension does not collect or process any of the following data:

Personal information

Browsing history

Form data

Authentication information

Payment or financial information

User activity or analytics

Cookies or local storage data

Anything typed, clicked, or viewed by the user

2. How the Extension Works

The extension runs entirely in the userâ€™s browser.
It only:

Reads HTML of the element the user clicks

Generates selectors locally

Displays the result directly to the user

No data leaves the device.

3. No Data Sharing

We do not sell, share, or transfer data to any third party.

4. No Remote Servers

The extension does not communicate with external servers.
All functionality is local and offline.

5. Permissions Explanation

The <all_urls> and activeTab permissions are required solely to allow the extension to:

Inspect elements on any webpage

Generate CSS/XPath selectors

Run scripts only when the user activates the extension

The extension never collects or transmits data from these permissions.
